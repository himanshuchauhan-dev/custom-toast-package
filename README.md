# Custom React Toast Package

A customizable React toast notification package with support for Success, Error, and Warning toasts.

## 🚀 Features

- Success Toast
- Error Toast
- Warning Toast
- Auto Dismiss Option
- Custom Duration
- Multiple Toast Positions
- Clear All Toasts
- Responsive UI

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS
